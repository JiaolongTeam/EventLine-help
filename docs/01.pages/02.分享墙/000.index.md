::: tip 前言

有用户会问类似的问题，这个功能是不是这么用的？
其实没有标准答案。我也只能给出我的用法，每个人有每个人的记录需求和习惯。工具是用来帮助思考的，而非限制思考的。**不要让自己适应工具**，要让工具适合你。

本页会收录部分用户的使用方法，你可以在这里找找灵感，也欢迎联系我投稿。

:::
